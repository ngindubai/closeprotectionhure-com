Event security in Belgrade is governed by Serbia's Law on Private Security Activity (Official Gazette of RS No. 104/2013) framework. Belgrade hosts corporate, financial, and sector-specific conferences at Belgrade Fair (Beogradski Sajam) and Crowne Plaza Belgrade infrastructure. Pre-event assessment, delegate close protection, and vetted airport transfers from BEG are the core components of the Belgrade event security programme.

## Serbia regulatory framework

All event security personnel at Belgrade events hold current Serbia credentials under Law on Private Security Activity (Official Gazette of RS No. 104/2013). Serbian Ministry of Interior (Ministarstvo unutrasnjih poslova, MUP) provides oversight.
