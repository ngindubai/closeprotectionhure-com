Event security in Prague is governed by Czech Republic's Act No. 69/2006 Coll. on Security Activities framework. Prague hosts corporate, financial, and sector-specific conferences at Prague Congress Centre (PCC) and Mandarin Oriental Prague infrastructure. Pre-event assessment, delegate close protection, and vetted airport transfers from PRG are the core components of the Prague event security programme.

## Czech Republic regulatory framework

All event security personnel at Prague events hold current Czech Republic credentials under Act No. 69/2006 Coll. on Security Activities. Czech Ministry of the Interior (Ministerstvo vnitra) provides oversight.
