## Philippine regulatory compliance

Manila event security companies hold current SOSIA company licences; individual personnel hold PNP Security Guard Licences. Armed personnel hold PNP firearm authorisations. PNP district command notification is confirmed for events above the statutory threshold.

## Manila event security environment

Advance work covers SMX, PICC, and BGC hotel venue assessments, KFR-aware departure management, terrorism awareness for crowded-space venues, PNP co-ordination, and armed vs. unarmed force posture determination based on event attendee profile.
