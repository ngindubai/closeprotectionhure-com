## Venue assessment and access control

A written venue security assessment is the starting point for any event security engagement in Los Angeles. The assessment addresses access and egress constraints, CCTV coverage, crowd-density planning for the planned attendance, and emergency evacuation routing. Access control operates on a tiered credentialling system with licensed personnel at all perimeter points.

## VIP protection and transport integration

Named principals and VIPs attending events in Los Angeles require individual close protection plans integrated with the overall event security structure. Security drivers on pre-planned routes coordinate arrival and departure with the event security manager to reduce the exposure that predictable vehicle staging creates.
