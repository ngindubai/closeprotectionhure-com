Edinburgh event security operates under the UK's Private Security Industry Act 2001, with all operators licensed by the Security Industry Authority (SIA), covering principal venues including the Edinburgh International Conference Centre (EICC), Assembly Rooms Edinburgh, and Murrayfield Stadium. Core service components span SIA-licensed access control, delegate close protection, and transfer management from Edinburgh Airport (EDI). Edinburgh's August Festival and Fringe, which brings approximately one million visitors to the city, creates Europe's largest concentrated public event environment outside major sporting competitions, requiring event security planning that accounts for elevated street-level crowd density across the Old Town and Royal Mile in addition to standard venue security protocols.

## UK regulatory framework

All event security personnel at Edinburgh events hold valid SIA licences under the Private Security Industry Act 2001. The Security Industry Authority (SIA) issues Door Supervisor (DS), Security Guard (SG), and Close Protection (CP) licences, each requiring enhanced DBS checks and sector-specific training. Police Scotland (emergency 999, non-emergency 101) coordinates public security at EICC, Murrayfield, and major Festival venues. MI5 assesses the UK terrorism threat as SUBSTANTIAL (2026), which informs event security planning for large public gatherings and international congresses.
