Event security in Madrid must be planned with this threat level in mind: vehicle exclusion, enhanced access control, and close coordination with the Policia Nacional are baseline requirements for large events.

Under Ley 5/2014 de Seguridad Privada, all commercial security personnel must hold DGP licences.
