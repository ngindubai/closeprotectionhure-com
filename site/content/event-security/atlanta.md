Atlanta is a major corporate event destination, hosting events at State Farm Arena, Mercedes-Benz Stadium, and the Georgia World Congress Center. Event security in Atlanta is regulated under OCGA Title 43, Ch 38, requiring all deployed personnel to hold current licences from the Georgia Secretary of State. ## Venue assessment and access control

A written venue security assessment is the starting point for any event security engagement in Atlanta. The assessment addresses access and egress constraints, CCTV coverage, crowd-density planning for the planned attendance, and emergency evacuation routing. Access control operates on a tiered credentialling system with licensed personnel at all perimeter points.

## VIP protection and transport integration

Named principals and VIPs attending events in Atlanta require individual close protection plans integrated with the overall event security structure. Security drivers on pre-planned routes coordinate arrival and departure with the event security manager to reduce the exposure that predictable vehicle staging creates.
