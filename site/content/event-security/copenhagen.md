Event security in Copenhagen is governed by Denmark's Security Guard Act (Vagtvirksomhedsloven) framework. Copenhagen hosts corporate, financial, and sector-specific conferences at Bella Center (Copenhagen Exhibition and Congress Centre) and Hotel d'Angleterre Copenhagen infrastructure. Pre-event assessment, delegate close protection, and vetted airport transfers from CPH are the core components of the Copenhagen event security programme.

## Denmark regulatory framework

All event security personnel at Copenhagen events hold current Denmark credentials under Security Guard Act (Vagtvirksomhedsloven). Danish Police (Rigspolitiet) and Danish Security and Intelligence Service (PET) provides oversight.
