Event security in Oslo is governed by Norway's Security Services Act (Vaktvirksomhetsloven, 2018) framework. Oslo hosts corporate, financial, and sector-specific conferences at Oslo Spektrum and The Thief Oslo infrastructure. Pre-event assessment, delegate close protection, and vetted airport transfers from OSL are the core components of the Oslo event security programme.

## Norway regulatory framework

All event security personnel at Oslo events hold current Norway credentials under Security Services Act (Vaktvirksomhetsloven, 2018). Norwegian Security Association (NSR) and Norwegian Police Security Service (PST) provides oversight.
