Event security in Athens is governed by Greece's Presidential Decree 78/2011 on Security Guards Licensing framework. Athens hosts corporate, financial, and sector-specific conferences at Metropolitan Expo Athens (Spata, adjacent to ATH) and Hotel Grande Bretagne Athens infrastructure. Pre-event assessment, delegate close protection, and vetted airport transfers from ATH are the core components of the Athens event security programme.

## Greece regulatory framework

All event security personnel at Athens events hold current Greece credentials under Presidential Decree 78/2011 on Security Guards Licensing. Hellenic Police (Elliniki Astynomia) and Ministry of Citizen Protection (YPDM) provides oversight.
