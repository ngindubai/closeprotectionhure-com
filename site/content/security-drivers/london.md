## SIA and TfL dual qualification

London's regulatory framework requires security drivers to hold both a TfL Private Hire Vehicle (PHV) driver licence for the commercial transport function and an SIA licence for any security element they perform. The SIA licence category depends on the security function: Door Supervisor for protective presence, Close Protection for a full CPO role combined with driving. Operators who provide security drivers should be able to confirm both qualifications for any proposed driver.
