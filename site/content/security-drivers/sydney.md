## NSW licensing and compliance

Service NSW through the Security Licensing and Enforcement Directorate (SLED) administers the security licensing framework in New South Wales. Security driver licences are issued under the Security Industry Act 1997. The relevant licence class depends on the security function performed; class 1E (Bodyguard) is the appropriate licence for integrated close protection driving. SLED licence verification is the primary due diligence step for any Sydney security driver engagement.
