What our event security covers

Philippine PNP and SOSIA Event Security Regulation

Event security in the Philippines is regulated under Republic Act 5487 (Private Security Agency Law) as amended, administered through the PNP Supervisory Office for Security and Investigation Agencies (SOSIA). Security companies must hold current SOSIA company licences; individual security guards must hold PNP Security Guard Licences issued after completion of the PNP Security Guard Basic Training Course and background verification. For events above defined attendance thresholds or involving high-profile principals, organisers are advised to notify the relevant Philippine National Police district command and co-ordinate with PNP for security support. Armed event security personnel require separate PNP firearm authorisations. SOSIA company licence and individual PNP Security Guard Licence should be confirmed before deployment.

Kidnap-for-Ransom Awareness at Manila Events

The kidnap-for-ransom (KFR) threat that defines Manila's general security environment is directly relevant to corporate event security: OSAC Manila Crime and Safety Report (2024) rates Manila as critical for KFR risk, with criminal groups surveilling hotel entrances, known corporate event venues, and predictable event departure times to identify and target foreign nationals and wealthy Filipino principals. Event security in Manila incorporates KFR awareness as a standard planning input: access control at event venue entrances, monitoring of the approach environment for surveillance activity, and departure management that prevents the simultaneous, visible departure of multiple high-value principals in an uncoordinated way. This is not a general crime measure: it is specific to the Manila KFR threat environment documented in OSAC.

Manila Principal Event Venues

Manila's principal event venues include: SMX Convention Center Manila (adjacent to SM Mall of Asia in Pasay) for large-scale international conferences; the Philippine International Convention Center (PICC) in Pasay for major government-adjacent and intergovernmental events; the Manila Hotel on Rizal Park (for state and high-profile corporate functions); BGC's Shangri-La at the Fort and Bonifacio High Street private event facilities for premium corporate functions; and the Conrad Manila for luxury private events on Manila Bay. Venue security assessment for Manila events confirms: the in-house security infrastructure at each property, PNP district command co-ordination status for notifiable events, and the KFR-specific event security protocols appropriate to each venue's exposure profile.

Terrorism and Civil Unrest Planning

Abu Sayyaf and IS-Philippines groups operate primarily in Mindanao and the Sulu archipelago; FCDO advises against all travel to western and central Mindanao. Their Manila threat is assessed against symbolic, crowded targets: shopping malls, major transport hubs, and government buildings are the primary assessed urban target categories. SMX Convention Center's proximity to SM Mall of Asia creates a specific advance assessment requirement for events at that venue. Political demonstrations in Metro Manila occur regularly near the Presidential Palace in Malacanang, Congress in Quezon City, and the Makati CBD; advance intelligence on demonstration schedules affecting event access routes is a standard planning input.

Armed Event Security and Force Posture

Armed event security is more prevalent at Manila events than in most comparable Asia-Pacific markets, reflecting the documented KFR threat and the availability of PNP-licensed armed CPOs. For events with high-profile foreign executive attendees, senior government officials, or HNWI principals with known Manila threat profiles, armed event security personnel are the appropriate deployment. For standard corporate conference security at established hotel venues with their own in-house security infrastructure, unarmed event security supplementing the venue's provision may be sufficient. The event security brief determines the appropriate force posture for each engagement; armed security is not a default for all Manila events, but the KFR risk context means it is assessed on an event-by-event basis in a way not required in lower-threat cities.
