Israeli RAAS Event Security Regulation

Event security in Israel is regulated by the Regulatory Authority for Private Security Services (RAAS) under the Ministry of Public Security. Security companies must hold RAAS company licences; individual security personnel must hold RAAS personal licences at the appropriate level. For public events above defined attendance thresholds, Israeli law requires event organisers to co-ordinate security plans with Israel Police and, for events above higher thresholds, obtain Israel Police approval for the security plan. Armed event security personnel require RAAS firearms permits authorised through Israel Police. The Israeli event security market has a deep pool of RAAS-licensed operators with IDF, police, and intelligence backgrounds, which is the quality differentiator in the Israeli market given the conflict environment.

Conflict Environment Event Security Context

FCDO advises against all travel to Israel. The US State Department rates Israel at Level 4. Any event in Tel Aviv as at June 2026 takes place in an active conflict environment with ongoing Hamas rocket fire from Gaza, the fragile Hezbollah ceasefire on the northern border, and the documented threat of surface attacks including the February 2025 Tel Aviv bus bombings. Event security in this environment goes beyond standard crowd management and access control: shelter protocols are integrated into every event plan (90 seconds to shelter from Tzeva Adom rocket alert siren), armed security personnel are standard rather than optional, and evacuation planning accounts for the possibility of a simultaneous external threat during the event.

Rocket Alert and Shelter Protocol Integration

Every Tel Aviv event security plan must integrate the Red Alert (Tzeva Adom) shelter protocol. When a Tzeva Adom siren sounds, all personnel and event attendees must reach a designated miklat (shelter) or reinforced room within 90 seconds. Event security advance work for Tel Aviv identifies: the designated miklat locations within and adjacent to each event venue, the capacity of each shelter relative to the expected event attendance, the route from every event space to the nearest shelter, and the communication protocol for event security staff to initiate shelter movement at siren activation. This protocol is briefed to all event security staff and venue personnel before the event, and confirmed to event organisers as a planning requirement.

Tel Aviv Event Venue Assessment

Tel Aviv's principal event venues include: the Tel Aviv Convention Center (Ganei HaTa'arukhot) in northern Tel Aviv for large-scale conferences, the David InterContinental Tel Aviv and the Norman Tel Aviv on HaYarkon Street for luxury private functions, and Dan Panorama Tel Aviv and the Sheraton Tel Aviv for corporate conference events. Venue security advance work for each property covers: confirmation of the in-house security infrastructure at each location (Israeli hotel security is generally highly developed post-2000s), the location and capacity of on-site miklat facilities, vehicle approach management for the HaYarkon beachfront venues (subject to specific approach management because of beach access and public promenade exposure), and coordination with venue security management on armed event security integration.

Surface Attack and Access Control Protocols

The February 2025 Tel Aviv bus bombings and the vehicle ramming attack history across Israeli urban environments create specific access control requirements for Tel Aviv events. Vehicle access management at Tel Aviv event venues goes beyond standard access control: vehicle barriers or managed vehicle exclusion zones for venues with public road frontage, pedestrian access screening proportionate to the event profile and RAAS security plan, and armed security personnel positioned to respond to surface attack scenarios. Vendor and service provider vetting for Tel Aviv events addresses supply chain security: catering, AV, and logistics providers are subject to enhanced vetting consistent with the conflict environment access control standard.
