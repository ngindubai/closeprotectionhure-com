The Problem: 6 Images Across 33 Pages

Six generic stock images rotate across 33 pages via modulo. Lagos gets the same hero as Karachi. The bodyguard page shares imagery with the event security page on list cards. Every repeat signals "template site" to both Google and visitors.

Image Optimisation Notes

  • Convert all JPGs to WebP before deploying (Hugo image processing or cwebp CLI).
  • Target file size: under 200KB per image after compression.
  • Hero video: hero-globe.mp4 is 161MB at 4K. Create a 720p web version (~5-10MB) before launch:
ffmpeg -i hero-globe.mp4 -vf scale=1280:720 -c:v libx264 -crf 28 -preset slow hero-globe-web.mp4
  • Thumbnail crops: Consider creating 400x300 crops of city images for the homepage grid cards.
